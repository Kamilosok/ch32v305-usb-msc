# Third Party Code

This project includes code from WCH (CH32V307 EVT package) in the `/platform` and `/vendor` directories.

- Source: <https://www.wch.cn/downloads/CH32V307EVT_ZIP.html>
- Some files in the [openwch](https://github.com/openwch/ch32v307) GitHub repository, which seems to include the EVT package, include Apache-2.0 headers, unlike code from the website.
- Other files do not include explicit license information.

These files are included as-is. Users should verify licensing terms before reuse.
